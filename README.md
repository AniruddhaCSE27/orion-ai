# 🚀 ORION AI --- Autonomous Research Engine

Transforming queries into structured intelligence using multi-agent AI.

## 🌟 Features

-   Multi-Agent System (Planner → Researcher → Writer)
-   Real-time web search (Tavily API)
-   LLM-powered reports (OpenAI)
-   Premium UI (Streamlit)
-   PDF export

## 🛠 Tech Stack

-   Frontend: Streamlit
-   Backend: FastAPI
-   LLM: OpenAI
-   Search: Tavily

## 🚀 Run Locally

``` bash
git clone https://github.com/AniruddhaCSE27/orion-ai.git
cd orion-ai
pip install -r requirements.txt
streamlit run app.py
```
